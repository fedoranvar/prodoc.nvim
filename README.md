# prodoc.nvim

a neovim comment and annotation plugin using coroutine.


## TODO

## Usage

```vim
nnoremap gcj :ProDoc<CR>
nnoremap gcc :ProComment<CR>
xnoremap gcc :ProComment<CR>
```
